# Changelog

## v1.0 — 2026-06-28 · Primera publicación

- App web interactiva completa para la 58ª Feria de Ciencias · Instituto Puertas del Sol.
- Secciones: Mapa, Agenda, Buscar, Guardados y Cómo llegar.
- Datos cargados: Inicial (Salas 2, 3, 4, 5), Primario (1° a 6° grado), Secundario (1° a 6° año).
- Docentes del secundario incluidos en cada ficha.
- Diseño mobile-first, todo embebido en un solo `index.html`.

### Pendiente para próximas versiones
